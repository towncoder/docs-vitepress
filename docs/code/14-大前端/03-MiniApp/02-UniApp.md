## quick start

### 利用 vue 脚手架新建项目

```shell
npm install -g @vue/cli 
vue create -p dcloudio/uni-preset-vue uniapp-init
```

选择模板，默认模板支持使用 TS ，本着完事从 Hello World 我们先选 [hello uni-app](https://github.com/dcloudio/hello-uniapp) 体验一下。
