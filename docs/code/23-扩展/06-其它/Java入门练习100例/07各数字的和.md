## 题目描述

<font color=green>难度：简单</font>

>计算给定整数12345的各位上数字的和。
>

## 知识点

- 除法运算
- 取余运算

## 解题思路

解题的关键在于如何拿到各个位上的数字。

举例：拿到34的个位和十位

```java
int a=34;
//整除运算，拿到3
int b=34/10;
//返回余数4
int c=34%10;
```

## 代码实现

```java
/**
 * 计算给定整数12345的各数字的和。
 */
public class question_07 {
    public static void main(String args[]) {
        int y = 12345;
        int r = 0 ;
        int sum = 0;
        while(y!=0) {
            r = y % 10;
            sum += r;
            y = y / 10;
        }
        System.out.println("y = " + sum);
    }
}
```
