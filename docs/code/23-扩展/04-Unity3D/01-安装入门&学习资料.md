## 像素地鼠

### 初始化场景

创建一个 2D 项目，选择保存位置和项目名

导入 PlayMaker ，是收费的，不过有[免费下载地址](https://www.windowszj.net/pcsoft/wangluo/47883.html)，下载好之后需要解压，拖到 `Project` 下的 `Assert` 文件夹内，点击 `Import`。
